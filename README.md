# PollyGlot AI - Translation App (V1)

PollyGlot is a web-based translation application that leverages Artificial Intelligence to provide accurate and context-aware translations.

## 🚀 Features
- **AI-Powered:** Uses OpenAI's GPT models for high-quality translations.
- **Minimalist UI:** Clean, easy-to-use interface built with Vanilla JS.
- **Multi-language:** Currently support French, Spanish, German, and Japanese.
- **Secure Backend:** Express server to handle API calls and protect environment variables.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Node.js, Express
- **AI Engine:** OpenAI API (Chat Completions)

## ⚙️ Installation & Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Create a `.env` file and add your `OPENAI_API_KEY`.
4. Run the server: `node server.js`.
5. Open `index.html` in your browser.
